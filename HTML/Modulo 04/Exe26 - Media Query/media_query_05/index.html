Exibindo conteúdos dos Menus

Conteúdos desenvolvidos em HTML precisam ser adaptados a diferentes tipos de telas. Uma boa solução para o problema é se propor a utilizar os Midas Quiries.

Midia Query é uma tecnologia presente no CSS, como ele é possível de definir padrões de valores utilizando-se uma condição para determinada configuração. Exemplo: se desenvolver uma página tanto para tela maior como as de nootebooks, como para telas menores (smartphones), será necessário informar um pixel minímo ou máximo (depende do que realmente precisar).

media screen and (max-width: 375px) {
          main {
              width: 768px;
          }
     }

Nesse código acima, o meu elemento "main" terá uma largura máxima de 768px se por acaso a largura do dispositivo que está executando esse HTML for no máximo 375px.
